# EduHome Gamified Tools 🎮

This folder contains interactive, gamified tools designed for the EduHome presenter dashboard and student engagement.

## Components

### 1. Student Score Dashboard (`index.html`)
An all-in-one management dashboard for tracking student performance across chapters.
- **Milestone Tracking**: Visual alerts when students reach defined score targets.
- **Interactive Leaderboard**: Real-time ranking with vibrant visual feedback.
- **Random Student Spinner**: A high-energy tool to select a student for interaction or rewards.
- **PDF/Image Slideshow**: Seamlessly present class materials alongside the scoring system.
- **Persistence**: Data is saved locally and can be imported/exported via Excel (XLSX).

### 2. Galactic Loot Wheel (`wheel 2.html`)
A standalone "Rewards Wheel" (Game Loot Edition) for incentivizing achievements.
- **Loot Pool**: Rewards range from "Common Dust" to "Legendary Gear".
- **Dynamic Slots**: Adjustable number of wheel sectors.
- **High Fidelity**: Orbitron typography, neon animations, and dramatic sound effects.

## Usage
These tools are built using vanilla JavaScript, Tailwind CSS (local copy), and various CDNs for confetti and Excel support. They can be run directly in any modern web browser.
